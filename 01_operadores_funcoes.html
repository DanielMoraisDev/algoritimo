Funções dos operadores!

  • == : igual à, retorna true caso os dados contenham o mesmo conteúdo;
  • != : diferente de, retorna true caso os dados sejam diferentes;
  • === : identico à, verifica estritamente igual o tipo e valor e caso passe ele fica true ;
  • !== : diferentão à, estritamente diferente o tipo e valor;
  • > : maior que, caso seja maior retorna true;
  • < : menor que, caso seja menor retorna true;
  • <= : maior e igual à, caso seja maior e igual á retorna true;
  • >= : menor e igual à, caso seja menor e igual á retorna true.
